try语句

try语句是一种用于捕获和处理异常的结构。通过try语句，我们可以编写代码来处理可能发生的异常情况，从而改善程序的稳定性和可靠性

try语句类似于if语句，若try语句内代码执行成功则不会执行catch语句内的代码

return语句

return语句用于从当前方法中返回一个值，并终止该方法的执行。

    {

        {
            Console.WriteLine("请输入需要使用的运算方式;");
            string FS = Console.ReadLine();//变量名取自"方式"的拼音首字母
            
            if (FS == "加法")
            {
                加法();
            }
            else if (FS == "减法")
            {
                减法();
            }
            else if (FS == "乘法")
            {
                乘法();
            }

            {
                除法();
            }

            {

            }
            else
            {

            Console.WriteLine("按C键继续，按其他键退出:");
            PD = Console.ReadLine();
            
        }
        
    }
